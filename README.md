# Algorithms Coursework App (Python)

## How to run

1. Ensure you are in the project folder:
   ```bash
   cd algorithms_app
   ```

2. Run the GUI:
   ```bash
   python app.py
   ```

## Tests (functional + basic performance)

### Functional tests (unittest)
```bash
python -m unittest discover -s tests -p "test_*.py"
```

### Performance scripts
```bash
python performance/perf_sorts.py
```

## Algorithms included
- RSA encryption/decryption (from scratch)
- Fibonacci using dynamic programming
- Selection sort / Bubble sort
- Merge sort (divide and conquer)
- Fisher-Yates shuffle of a standard 52-card deck
- Recursive factorial
- Array statistics: min, max, mode, median, Q1, Q3
- Palindromic substring counter using memoization
